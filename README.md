## E-Commerce Delivery Delay Analysis

### Overview
Analysis of delivery delays using the Olist Brazilian E-commerce dataset to identify order failures and logistics bottlenecks.

### Key Insight
The **shipped → delivered** stage is the primary contributor to late deliveries and failed orders.

### Tools
Python (Pandas), Power BI

### Deliverables
- Master dataset  
- Power BI dashboard  
- Final report

### ML Outcome
The XGBoost model achieved a test log loss of 0.30, demonstrating well-calibrated probability predictions and strong generalization performance. The close alignment between training and test log loss indicates minimal overfitting.
